## 📌 공통 입력 조건

모든 테스트 케이스는 동일한 입력 조건을 사용합니다:

- **생리 기록**: 
  - 2025-01-01 ~ 2025-01-05
  - 2025-01-26 ~ 2025-01-30 (25일 주기)
  - 2025-02-20 ~ 2025-02-24 (25일 주기)
- **배란 테스트 기록**: 없음
- **배란일 직접 입력**: 없음
- **피임약 패키지 시작일**: 없음
- **오늘 날짜**: 2025-03-15
- **생리 주기 설정**: 
  - 평균 주기: 25일 (실제 계산)
  - 평균 기간: 5일 (실제 계산)
  - 주기 길이 범위: 25-25일
  - 수동 평균 주기: 25일
  - 수동 평균 기간: 5일
  - 자동 계산 사용: false
- **피임약 설정**:
  - 피임약으로 계산: false

**⚠️ 주요 차이점**: 25일 주기는 26일 미만이므로 다른 계산 공식 적용
- 배란기: period - 16 ~ period - 14 = 9~11 (인덱스 9~11, 시작일 + 9 ~ 시작일 + 11)
- 가임기: period - 19 ~ period - 11 = 6~14 (인덱스 6~14, 시작일 + 6 ~ 시작일 + 14)
