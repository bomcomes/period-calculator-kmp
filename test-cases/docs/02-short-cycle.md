# 짧은 주기 (25일) 테스트 케이스

## 📌 공통 입력 조건

모든 테스트 케이스는 동일한 입력 조건을 사용합니다:

- **생리 기록**: 
  - 2025-01-01 ~ 2025-01-05
  - 2025-01-26 ~ 2025-01-30 (25일 주기)
  - 2025-02-20 ~ 2025-02-24 (25일 주기)
- **배란 테스트 기록**: 없음
- **배란일 직접 입력**: 없음
- **피임약 패키지 시작일**: 없음
- **오늘 날짜**: 2025-03-15
- **생리 주기 설정**: 
  - 평균 주기: 25일 (실제 계산)
  - 평균 기간: 5일 (실제 계산)
  - 주기 길이 범위: 25-25일
  - 수동 평균 주기: 25일
  - 수동 평균 기간: 5일
  - 자동 계산 사용: false
- **피임약 설정**:
  - 피임약으로 계산: false

**⚠️ 주요 차이점**: 25일 주기는 26일 미만이므로 다른 계산 공식 적용
- 배란기: period - 16 ~ period - 14 = 9~11일차 (인덱스 8~10)
- 가임기: period - 19 ~ period - 11 = 6~14일차 (인덱스 5~13)

- **생리 지연일**: 없음
- **배란기들**: 없음 (조회 기간 내 배란기 없음)
- **가임기들**: 없음 (조회 기간 내 가임기 없음, 다음 가임기는 2025-03-18부터)

## TC-02-03: 1개월 조회 (생리 사이)
### 📌 테스트 목적
실제 기록된 두 생리 기간 사이의 구간을 조회할 때 배란기/가임기가 정확한지 검증하고, 다음 생리 정보도 함께 제공되는지 확인

### 📝 입력 조건
- **조회 기간**: 
  - 시작: 2025-01-31
  - 종료: 2025-02-19

### ✅ 예상 결과

**주기 개수: 2개**

**주기 1 (pk=2):**
- **실제 생리 기록**: 2025-01-26 ~ 2025-01-30 (두 번째 생리)
- **생리 예정일들**: 없음 (Period 3는 실제 기록, 예정일이 아님)
- **생리 지연일**: 없음
- **배란기들**: 
  - 2025-02-03 ~ 2025-02-05 (생리 시작 9-11일차)
- **가임기들**: 
  - 2025-01-31 ~ 2025-02-08 (생리 시작 6-14일차)
- **주기**: 25일

**주기 2 (pk=3):**
- **실제 생리 기록**: 2025-02-20 ~ 2025-02-24 (세 번째 생리, 다음 주기 정보 제공)
- **생리 예정일들**: 없음 (조회 범위 밖)
- **생리 지연일**: 없음
- **배란기들**: 없음 (조회 범위 밖)
- **가임기들**: 없음 (조회 범위 밖)
- **주기**: 25일

### 🔍 검증 포인트
- [ ] 실제 생리 기록 사이 구간의 배란기/가임기가 짧은 주기 공식(9-11일차, 6-14일차)으로 정확히 계산되는가 (pk=2)
- [ ] 생리 사이 구간의 배란기가 9-11일차로 계산되는가 (pk=2)
- [ ] 생리 사이 구간의 가임기가 6-14일차로 계산되는가 (pk=2)
- [ ] 다음 생리 정보(pk=3)가 함께 반환되어 주기의 끝을 알 수 있는가
- [ ] pk=3의 배란기/가임기는 조회 범위 밖이므로 비어있는가
