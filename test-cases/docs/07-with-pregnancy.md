# 임신 모드 테스트 케이스

## 그룹 1: 임신 모드 (생리 기록 입력 중 임신)

### 📌 공통 입력 조건

- 생리 기록:
  - 2025-01-01 ~ 2025-01-05 (pk=1)
  - 2025-02-01 ~ 2025-02-05 (pk=2)
- 배란 테스트 기록: 없음
- 배란일 직접 입력: 없음
- 생리 주기 설정:
  - 평균 주기: 31일 (실제 계산) → 평균 주기: 28일 (임신 중 기본값)
  - 평균 기간: 5일 (실제 계산)
  - 수동 평균 주기: 28일
  - 수동 평균 기간: 5일
  - 자동 계산 사용: true
- 피임약 설정:
  - 피임약으로 계산: false
- 임신 정보:
  - 출산 예정일: 2025-11-25 (사용자 입력)
  - 임신 시작일: 2025-02-18 (자동 계산: 출산 예정일 - 280일)
  - 마지막 생리 시작일: null
  - 예정일 확정 여부: true
  - 출산 완료: false
  - 유산 여부: false
