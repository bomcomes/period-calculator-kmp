## 계산 시나리오 예시

### 시나리오 1: 주기 1 - 피임약 복용 중
```
입력:
- 마지막 생리 시작일: 2025-01-05
- 평균 주기: 30일
- 피임약 복용: true
- 배란 테스트: 2025-01-18 양성
- 사용자 입력 배란일: 2025-01-19

예상 결과:
- 다음 생리 예정일: 2025-02-04
- 배란일: null (피임약 복용 중)
- 가임기: null (배란 없음)
```

### 시나리오 2: 주기 2 - 자연 주기
```
입력:
- 마지막 생리 시작일: 2024-12-06

- 피임약 복용: false
- 배란 테스트: 2024-12-19 양성

예상 결과:

- 배란일: 2024-12-20 (사용자 입력 우선)
- 가임기: 2024-12-15 ~ 2024-12-20
```
